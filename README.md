# Alex Styles Portfolio

## Description

This project is a modern portfolio website created for Alex Styles, a graphic designer who wants to showcase creative projects and attract potential clients.

## Features

* About Me section
* Portfolio section with project cards
* Contact form
* Responsive design for desktop, tablet, and mobile devices
* CSS animations and hover effects
* Interactive modal window
* Tooltips for navigation links

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* JavaScript

## Responsive Design

The website adapts to different screen sizes using media queries:

* Mobile devices (max-width: 768px)
* Tablets (max-width: 1024px)
* Desktop screens (min-width: 1025px)

## UI/UX Enhancements

* Hover effects on buttons and links
* Project modal popup
* Smooth transitions and animations
* Improved typography and color palette

## Microsoft Copilot Usage

Microsoft Copilot was used to:

* Generate CSS styles
* Create responsive layouts
* Add animations and transitions
* Improve UI/UX features
* Optimize and organize the code
